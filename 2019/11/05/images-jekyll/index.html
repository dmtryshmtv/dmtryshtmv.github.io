Customizing Embedded Images with Jekyll

Nov 5, 2019 • jekyll, site

Something I anticipate doing in blog posts is embedding images. Using the standard ![alt-text](link-to-image-asset) construction, we can do a basic embedding. However, what if we want to modify the size or the centering properties of the image?

The following instructions are based on the general information found here, but distilled for my specific usecase.

To add HTML attributes and CSS classes we should do the following ![alt-text](link-to-image-asset){:attribute="value" class="class-name"}. In the specific case of resizing an image this would be ![alt-text](link-to-image-asset){:width="400px" height="300px"}.

To add image alignment, we will have to modify our Jekyll theme with some custom CSS classes. This will vary for each theme, so check your theme’s documentation, but for the whiteglass theme, I had to make a new file main.scss in SITE-DIRECTORY/assets/ and fill it with the following:

To elaborate a little, we added the front matter, imported the standard theme CSS, and then added custom CSS classes below to define groups of properties that we can now apply to images. So to embed a centered image, for example, we would write ![alt-text](link-to-image-asset){:class="image-centered"}.
