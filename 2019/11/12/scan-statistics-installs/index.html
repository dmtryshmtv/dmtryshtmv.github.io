Installs for the Scan Statistics for Network Epidemics Project

Nov 12, 2019 • research-software

I’m going to go from scratch, assuming you’re on Mac OS X.

Install pyenv and virtualenv

This will manage your python installations.

brew install pyenv pyenv-virtualenv

Install your desired python, which in this case is 3.7.3.

pyenv install 3.7.3

Now navigate to your directory and create a new virtualenv for this directory.

pyenv virtualenv network_analysis

For future reference:

  • pyenv stores its virtualenvs in: /.pyenv/versions/
  • pipenv stores them in: /.local/share/

Install the basics

pip install numpy ipykernel scipy matplotlib scikit-learn networkx seaborn xlrd pillow blackcellmagic joblib netwulf pycodestyle

Install basemap

Warning: large download (80MBs).

pip install git+https://github.com/matplotlib/basemap.git

Install EoN

You need to get the 1.0.9rc1 version or older, which wasn’t on PyPi.

pip install git+https://github.com/springer-math/Mathematics-of-Epidemics-on-Networks.git

Install this virtualenv as an IPython kernel in your (assumed existing) Jupyter installation

python -m ipykernel install --user --name=network_analysis

Install graph-tool on OS X with pipenv

First install it on your machine using Brew (pip is unfortunately not available here, because graph-tool is a hybrid C++/Python package).

brew install graph-tool

Now this will install graph-tool into a Python that Brew deems best. Make sure that this is the same version as the one in your virtual-env.

Next, we will want to symlink the graph-tool installation into your virtual-env installation (yes, this is awful and completely defeats the purpose of using pipenv, but these are the wonders of dev code without which life would be so much less fun :|). Use the directories here for a hint of where to go for the right directories (use pipenv –venv to find the virtual env directory).

ln -s /usr/local/Cellar/graph-tool/2.29/lib/python3.7/site-packages/graph_tool/ ~/.pyenv/versions/network_analysis/lib/python3.7/site-packages

We also need pycairo for drawing (comes with the brew installation).

ln -s /usr/local/Cellar/py3cairo/1.18.1/lib/python3.7/site-packages/cairo/ ~/.pyenv/versions/network_analysis/lib/python3.7/site-packages
